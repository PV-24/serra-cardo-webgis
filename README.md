# Serra de Cardó WebGIS

Aquest repositori conté una pàgina web sobre la **Serra de Cardó**, un espai natural situat a l’entorn de Benifallet, a les Terres de l’Ebre. La web combina informació general, rutes de senderisme, serveis locals, consells de seguretat i un mapa interactiu elaborat amb QGIS.

## Objectiu del projecte

L’objectiu principal és crear una web informativa i cartogràfica que permeti conèixer millor la Serra de Cardó i planificar una visita a l’entorn. La pàgina està pensada com una eina útil per a persones interessades en el senderisme, el patrimoni natural i la descoberta del territori.

## Contingut de la web

La web inclou diferents apartats:

- **Inici**: presentació general de la Serra de Cardó.
- **Informació**: dades generals sobre l’espai, el paisatge i el seu valor natural.
- **Rutes**: descripció de diverses rutes de senderisme amb distància, dificultat, durada i desnivell.
- **Serveis**: restaurants, productes locals i activitats disponibles a Benifallet.
- **Seguretat**: consells bàsics per fer rutes de muntanya amb seguretat.
- **Mapa**: visor cartogràfic interactiu amb rutes i punts d’interès.
- **Contacte**: formulari de contacte de la web.

## Mapa interactiu

El mapa s’ha creat amb **QGIS** i s’ha exportat amb **qgis2web** en format **Leaflet**. Inclou diferents capes cartogràfiques, com rutes de senderisme, punts d’interès, miradors, ermites, coves, aparcaments i altres elements destacats. Els punts disposen de finestres emergents amb informació i imatges.
